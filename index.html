I have more than 10 years experience building software for clients all over the world. Below is a quick overview of my main technical skill sets and technologies I use. Want to find out more about my experience? Check out my online resume and project portfolio.

Vanilla Javascript

Vanilla JavaScript refers to plain JavaScript code that is not extended or modified by any additional libraries or frameworks. It is the purest form of JavaScript and is used to build web applications and interact with HTML and CSS.

React

React is a JavaScript library for building user interfaces. It allows developers to create reusable components for building efficient and scalable web applications. It was developed by Facebook and is widely used in the creation of modern web applications.

Node.js

Node.js is a Javascript runtime built on Chrome's V8 engine. It allows developers to run Javascript on the server, creating web applications and APIs. It is known for its high performance and asynchronous, event-driven architecture.

MongoDB

MongoDB is a popular, open-source NoSQL database that stores data in a flexible, JSON-like format called BSON. It is known for its scalability and high performance, making it ideal for handling large amounts of data.
